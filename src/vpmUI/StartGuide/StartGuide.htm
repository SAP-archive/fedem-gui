FEDEM™, an acronym for Finite Element Dynamics in Elastic Mechanisms, provides both
a technology platform and an engineering framework for virtual testing of complex mechanical assemblies.
It provides a complete set of features to create, solve and post-process a model in a 3D graphical environment.
Dynamics results in the form of curves and animations are available during and after running the model solution process.
